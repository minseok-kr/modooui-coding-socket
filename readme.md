# 모두의 코딩
## Intro
실시간 코드 스터디 플랫폼, 모두의 코딩입니다.

## Purpose
오프라인에서 이루어지는 코드 스터디에서는 흔히 발생하는 문제들이 존재합니다. 스터디 주최자와 참여자 간의 소통의 어려움이나 각 스터디원에게의 피드백 전달의 어려움 등이 그런 것들이죠. 이 프로젝트는 그 어려운 점들을 개선시켜보고자 만든 실시간 코드 스터디 플랫폼입니다.
